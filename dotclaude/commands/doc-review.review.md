# Review: doc-review.md

> Reviewed: 2026-01-22
> Original: dotclaude/commands/doc-review.md

## 概要 (Summary)

任意のドキュメントファイルをレビューし、`<filename>.review.md`として出力するコマンドの仕様書。目的と役割、完全性、明確性、一貫性、技術的正確性、改善点の観点から分析し、テンプレートに基づいてレビューファイルを生成する。単一ファイルおよびglobパターンによる複数ファイル指定に対応。

## 評価 (Evaluation)

### 品質 (Quality)

- [x] **完全性 (Completeness)**: 必要な情報が網羅されている
- [x] **明確性 (Clarity)**: 読者にとって分かりやすい
- [x] **一貫性 (Consistency)**: 用語・スタイルが統一されている

### 技術的正確性 (Technical Accuracy)

- [x] **情報の正確性 (Correct information)**: 記載内容が正確
- [x] **最新性 (Up-to-date content)**: 情報が最新の状態

## 総評 (Overall Assessment)

汎用的なドキュメントレビューコマンドとして、シンプルで使いやすい設計。6つの分析観点が明確に定義されており、テンプレートベースの出力で一貫性のあるレビュー品質を確保できる。globパターンによる複数ファイル対応も明記されており、バッチ処理も可能。テンプレート参照パスも`~/.claude/templates/`と`dotclaude/templates/`の両方に対応している。日本語出力のルールも適切だが、技術用語の扱いについて補足が望ましい。
