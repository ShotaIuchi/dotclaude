> Reviewed: 2026-01-24

## 概要 (Summary)

Claude Code のコマンド（`.claude/commands/*.md`）の書き方を解説する包括的なガイドドキュメント。コマンドの基本構造から、引数処理、サブコマンド、動的コンテキスト注入、スキルとの違いまで幅広くカバーしている。コマンド作成者が必要とする情報を網羅的に提供することを目的としている。

## 総評 (Overall Assessment)

本ドキュメントは、Claude Code コマンドの作成に必要な情報を体系的にまとめた優れたリファレンスガイドである。基本構造から応用的な機能まで段階的に説明されており、初めてコマンドを作成するユーザーでも理解しやすい構成となっている。

特に評価できる点：
- コード例が豊富で、各機能の使い方が具体的に示されている
- 「コマンド vs スキル」の比較表により、適切な選択の判断基準が明確
- 実装チェックリストにより、コマンド作成時の確認事項が一覧化されている

改善が望まれる点：
- 動的コンテキスト注入の記法説明がやや分かりにくい
- 一部のフィールドや機能について、参照先ドキュメントへの誘導が不足

全体として、コマンド作成の実用的なガイドとして十分な品質を備えており、dotclaude プロジェクトの重要なリファレンスドキュメントとして機能している。上記の改善点を反映することで、さらに完成度の高いガイドになると考えられる。
