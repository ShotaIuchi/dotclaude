# ワークフローエージェント: research

## 概要

Issue の背景調査、コードベース分析、技術コンテキストの収集を行うワークフローエージェント。
`/wf1-kickoff` スキルから呼び出される。

## メタデータ

- **ID**: research
- **Base Type**: explore
- **Category**: workflow

## 機能

1. **Issue 分析** - Issue のタイトル、本文、ラベル、マイルストーンを解析
2. **コードベース調査** - Issue で言及されたファイルやモジュールを特定
3. **依存関係分析** - 影響を受けるモジュールや関連テストファイルを特定
4. **技術背景** - 使用中の技術スタックを確認、関連ドキュメントを収集

## 制約

- 読み取り専用（調査中にコードを変更しない）
- 機密情報（.env、credentials 等）は読み取らない
- 最大調査時間: 10分
- 詳細分析対象ファイル数上限: 50ファイル

## 出力

調査結果は `.wf/research/<issue_number>.md` に保存され、Kickoff 作成の入力として使用される。

## 使用方法

```
/agent research issue=123
```

または `/wf1-kickoff` 経由で自動実行。
