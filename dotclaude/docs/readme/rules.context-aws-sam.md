# AWS SAM コンテキストルール

AWS SAMサーバーレス開発時に自動適用されるコンテキスト。

## 適用タイミング

以下の作業時に適用:
- Lambda関数作成
- SAMテンプレート（template.yaml）設計
- API Gateway連携設定
- sam localでのローカルテスト

## 参照ドキュメント

| ドキュメント | 目的 |
|-------------|------|
| AWS Conventions | プロジェクト構造、命名 |
| SAM Architecture | Lambda/APIパターン |
