# Android コンテキストルール

Android開発タスク時に自動適用されるコンテキスト。

## 適用タイミング

以下の作業時に適用:
- Android機能実装（ViewModel、Repository、UseCase）
- Jetpack Compose UI作成
- Hilt依存性注入設定
- MVVM/UDFパターン適用

## 参照ドキュメント

| ドキュメント | 目的 |
|-------------|------|
| Clean Architecture | レイヤー分離の原則 |
| Testing Strategy | テストパターンとカバレッジ |
| Android Conventions | 命名、構造、イディオム |
| Architecture Patterns | MVVM/UDF実装 |
