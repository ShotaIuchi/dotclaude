# ワークフローエージェント: implementer

## 概要

Plan の1ステップを実装し、テスト実行とログ記録を行うワークフローエージェント。
`/wf5-implement` スキルから呼び出される。

## メタデータ

- **ID**: implementer
- **Base Type**: general-purpose
- **Category**: workflow

## 機能

1. **コード実装** - 計画に従ってファイルを作成/変更、既存のコードスタイルに準拠
2. **テスト実行** - ステップ完了条件に基づいてテストを実行し結果を記録
3. **実装ログ更新** - `04_IMPLEMENT_LOG.md` に変更を記録、次のステップへの引き継ぎ情報

## 制約

- **計画外変更禁止**: `02_PLAN.md` に記載されていない変更は行わない
- **1実行 = 1ステップ**: 複数ステップを一度に実装しない
- **テスト必須**: ステップ完了条件を満たすテストを実行
- **ログ必須**: `04_IMPLEMENT_LOG.md` に実装内容を記録

## 出力形式

実装完了レポートには以下を含む:

- ステップ情報（work-id、ステップ番号、タイトル）
- ファイル変更（パス、変更タイプ、概要）
- テスト結果
- 次ステップ情報
- 引き継ぎ用メモ

## 使用方法

`/wf5-implement` 経由で自動実行。
