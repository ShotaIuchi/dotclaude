# AWS SAM スキル

## 概要

AWS SAM によるサーバーレスアプリケーション開発、Lambda 関数実装、テンプレート設計、ローカルテスト、デプロイのためのスキル。
参照ドキュメントのパターンを読み込み、ユーザーの実装タスクに適用する。

---

## 使用場面

以下の場面で使用：

- AWS SAM サーバーレスアプリケーション開発
- Lambda 関数の実装
- SAM テンプレート設計
- ローカルテスト実行
- サーバーレスアプリケーションのデプロイ
