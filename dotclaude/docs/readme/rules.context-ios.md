# iOS コンテキストルール

iOS開発タスク時に自動適用されるコンテキスト。

## 適用タイミング

以下の作業時に適用:
- iOS機能実装（ViewModel、Repository）
- SwiftUIビュー作成
- async/awaitまたはCombine使用
- MVVMパターン適用

## 参照ドキュメント

| ドキュメント | 目的 |
|-------------|------|
| Clean Architecture | レイヤー分離の原則 |
| Testing Strategy | テストパターンとカバレッジ |
| iOS Conventions | 命名、構造、イディオム |
| Architecture Patterns | MVVM/状態管理 |

## 外部リファレンス

- Swift Concurrency
- SwiftUI
- Combine
