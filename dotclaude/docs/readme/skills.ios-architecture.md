# iOS Architecture スキル

## 概要

Apple 公式ガイドラインに基づいた SwiftUI + MVVM / 状態管理パターンのスキル。
参照ドキュメントのパターンを読み込み、ユーザーの実装タスクに適用する。

---

## 使用場面

以下の場面で使用：

- iOS 機能の実装
- SwiftUI View の作成
- ViewModel のセットアップ
- async/await または Combine の利用
- MVVM パターンの適用
