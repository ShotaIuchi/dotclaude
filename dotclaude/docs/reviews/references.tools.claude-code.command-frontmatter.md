# Review: command-frontmatter.md

> Reviewed: 2026-01-24
> Original: dotclaude/references/tools/claude-code/command-frontmatter.md

## 概要 (Summary)

Claude Code のコマンド（`.claude/commands/*.md`）およびスキル（`.claude/skills/*/SKILL.md`）で使用できるYAMLフロントマターのリファレンスドキュメント。フロントマターの基本構造、利用可能なフィールド一覧、各フィールドの使い方と実例を網羅的に解説している。

このドキュメントの主な役割：
- コマンド/スキル開発者向けの技術リファレンス
- フロントマターフィールドの仕様定義
- 実践的な使用例の提供

## 改善点 (Improvements)

### 優先度高 (High Priority)

| # | 箇所 | 問題 | 提案 | Status |
|---|------|------|------|--------|
| 1 | 参考リンク セクション | 外部ドキュメントへのリンクが2つのみで、公式ドキュメントの更新により古くなる可能性がある | リンク先の最終確認日を記載し、定期的な確認を促す注記を追加。または公式ドキュメントのバージョン情報を含める | ✓ Fixed (2026-01-24) |
| 2 | フロントマターフィールド一覧 | `external` フィールドの具体的な使用例がない | `references` と同様に、`external` フィールドの具体的な記述例を追加する | ✓ Fixed (2026-01-24) |

### 優先度中 (Medium Priority)

| # | 箇所 | 問題 | 提案 | Status |
|---|------|------|------|--------|

| 2 | allowed-tools | 利用可能なツール名の完全なリストがない | 使用可能なツール名一覧（Read, Glob, Grep, Bash, Write, Edit等）を別表または参照リンクとして追加 | ✓ Fixed (2026-01-24) |

## 総評 (Overall Assessment)

本ドキュメントは、Claude Code のコマンドおよびスキル開発に必要なフロントマター情報を適切にまとめた良質なリファレンスである。

**強み:**
- テーブル形式でのフィールド一覧が見やすく、検索性が高い
- 実践的なコード例が豊富で、即座に活用可能
- 日本語での解説により、日本語話者にとって理解しやすい
- コマンドとスキルの違いが明確に対比されている

**改善の余地:**
- 一部フィールド（`external`, `context`）の詳細説明が不足
- 外部リンクの鮮度管理が課題
- エラー時の挙動やエッジケースの記載がない

全体として、開発者がすぐに参照して実装に活かせる実用的なドキュメントであり、dotclaudeプロジェクトの参照ドキュメントとして適切な品質を満たしている。上記の改善点を順次反映することで、より完全なリファレンスとなる。
