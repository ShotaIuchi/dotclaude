# iOS Context Rule

Auto-applied context when working on iOS development tasks.

## When to Apply

Apply this context when:
- Implementing iOS features (ViewModel, Repository)
- Creating SwiftUI views
- Using async/await or Combine
- Applying MVVM patterns
